# Документація API

## Головна сторынка
-   **Мутод:** GET
-   **Route:** '/'
-   **Опис:** Повертаэ HTML головної сторінки
-   **Приклад Відповіді:** HTML Контерт
-   **Статус:** 200 OK

-   **Route:** '/login'
